# Laravel API

Prérequis :

-   installer composer
-   avoir un serveur en local (WAMP par exemple)

---

Lancement :

-   vérifier le .env
-   `composer install`
-   `php artisan serve`

---

Utils :

-   lancer les migrations : `php artisan migrate:fresh`
-   bdd recommandée : utf8mb4_general_ci
-   en cas de problème essayer : `composer dump-autoload`
